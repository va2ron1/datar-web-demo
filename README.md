
# DATAR Web Demo
Environment variables (.env)

    VUE_APP_DATAR_API_KEY=''
## Requirements

- [DATAR API Key](https://datar.online)
- Node.js v10 or later
## Install

```bash
npm install
```

## Usage

```bash
npm run serve
```
